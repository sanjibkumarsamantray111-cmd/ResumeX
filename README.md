# ResumeX – AI Resume Builder

ResumeX is a modern web application that helps users create professional resumes quickly using customizable templates and an intuitive interface.
The platform allows users to build, edit, and manage resumes easily.

## 🚀 Features

* Multiple resume templates
* Easy form-based resume creation
* Real-time preview of resume
* Responsive design (mobile friendly)
* Template switching
* Clean and modern UI

## 🛠 Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Vite

Backend:

* Node.js
* Express.js

Other Tools:

* Git & GitHub

## ⚙️ Installation

Clone the repository

```
https://github.com/sanjibkumarsamantray111-cmd/ResumeX.git```

Go to project folder

```
cd ResumeX
```

Install dependencies

```
npm install
```

Run the project

```
npm run dev
```
